# Notes-for-JupyterNotebook

Here are some notes regarding to the jupyter notebook:

Using markdown for jupyter notebook
Click on the code box, and click on the 'Cell' menu at the top of the screen. From there select 'Cell type' and click 'Markdown'.

![image](https://user-images.githubusercontent.com/54916616/174951815-3f6f060d-6aa5-432f-b086-9cde6c066763.png)

To make text bold enclose it between two asterisks:
** bold **

Or to italicise, use one asterisk
* italics *

Making titles (remember the space between # and the texts)
# Title

## Subtitle

### SubSubtitle

Writing maths
$ 1+1=2 $
$$ 2+2=4 $$
